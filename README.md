# Password Generator

A secure random password generator built with Python.

## Features
- Generate passwords 8-32 characters
- Includes letters, numbers, and symbols
- 5 attempts per session limit
- User-friendly menu interface

## Usage
```bash
python password_gen.py
```

## Technologies
- Python 3
- secrets module (cryptographically secure)
- string module
